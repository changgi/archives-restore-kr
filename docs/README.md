## 📖 문서를 읽는 순서

**내가 투자자/스폰서/관리자라면** → PLAN.md 만 읽어도 충분합니다.

**내가 개발자이고 코드를 이해하고 싶다면** → TECHNICAL.md → (필요 시) BUILD.md

**내가 이 프로젝트를 내 공공데이터에도 적용하고 싶다면** → BUILD.md 전체 + TECHNICAL.md §12 파일 구조

**내가 홍보를 도와주고 싶다면** → SOCIAL_CAROUSEL.md 만 보면 됩니다. 복붙 가능.
