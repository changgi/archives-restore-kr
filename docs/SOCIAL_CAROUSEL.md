## 8. 이미지 제작 가이드 (Figma/Canva 템플릿)

### 색상 팔레트

```
배경    #0F0F0F  (부 #1A1A1A)
텍스트  #F5F5F5  (부 #A0A0A0, 약 #6B6B6B)
포인트  골드 #D4A853
Before  레드 #C53030
```

### 타이포그래피

```
제목     Pretendard 800 · 72-104pt · 자간 -2
부제목   Pretendard 500 · 24-32pt · 행간 1.4
eyebrow  Pretendard 500 · 11-14pt · 자간 +6 · UPPERCASE
본문     Pretendard 400 · 16-18pt · 행간 1.6
```

### 공통 요소

- 좌상/우상/좌하/우하 **4코너 골드 L자** (길이 20-40px)
- 상단 중앙 **수직 골드 레일** (높이 80-120px, linear-gradient fade)
- 배경 **점 패턴** (32×32px 그리드, opacity 0.04)
- `┤ English Eyebrow ├` 구성 (divider 양쪽 40-80px 골드 가로선)

### Canva 빠른 템플릿

1. 새 디자인 → "Instagram 게시물 (정사각형)" 또는 "Carousel Post (4:5)"
2. 배경색 `#0F0F0F`
3. Elements → "Line" 으로 4코너 L자 (골드 `#D4A853`)
4. Text → Pretendard 800 큰 제목
5. Text → Pretendard 500 trackingMax "ENGLISH EYEBROW"
