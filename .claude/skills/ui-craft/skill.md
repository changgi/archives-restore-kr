# UI Craft — 인터랙티브 UI 컴포넌트 구현

## 디자인 철학
GitHub 스타 1만개를 달성한 프로젝트들의 공통점: **처음 3초의 임팩트**. 사이트에 들어온 순간 "이건 다르다"는 느낌을 주는 비주얼과 인터랙션이 핵심이다.

### 참고할 디자인 패턴
- 다크 기반 + 포인트 컬러 (전통/문화재 → 금색/적색 포인트)
- 대비감 있는 타이포그래피 (한글 Pretendard + 영문 Inter)
- 마이크로 인터랙션 (호버, 스크롤, 클릭 피드백)
- 스크롤 트리거 애니메이션 (요소가 뷰포트에 들어올 때)

## 기술 제약
- **Framer Motion 11**: 모든 애니메이션의 기본 라이브러리
- **Tailwind CSS 4**: 스타일링, CSS 변수 기반 테마
- **'use client'**: 인터랙티브 컴포넌트에만 적용
- **60fps 보장**: transform/opacity만 애니메이트, layout 트리거 금지

## 핵심 컴포넌트 구현 가이드

### 1. ImageCompareSlider
드래그/터치로 복원 전/후를 비교하는 슬라이더.
- 양쪽 이미지를 `clip-path`로 분할
- 드래그 핸들에 포인터 이벤트 바인딩
- 터치/마우스 모두 지원
- 이미지 로딩 시 스켈레톤 표시
- 접근성: 키보드 화살표로 슬라이더 이동 가능

### 2. TimelineView
연도별 복원 사례를 인터랙티브하게 보여주는 타임라인.
- 가로 스크롤 타임라인 (데스크탑) / 세로 스크롤 (모바일)
- 연도 노드 클릭 시 해당 연도 사례 펼침
- 연결선 SVG 애니메이션
- 사례 카드가 타임라인에서 확장

### 3. FilterGrid
카테고리/기관/유형별 필터링 + 레이아웃 애니메이션.
- `AnimatePresence`로 필터 전환 시 부드러운 재배치
- `layoutId`로 카드 위치 애니메이션
- 필터 버튼에 활성 개수 뱃지
- 결과 없음 시 일러스트 표시

### 4. GalleryLightbox
풀스크린 이미지 갤러리.
- Masonry 레이아웃 그리드
- 클릭 시 풀스크린 오버레이 (layoutId transition)
- 스와이프로 이전/다음 (터치)
- 핀치 줌 지원
- ESC/배경 클릭으로 닫기
- 키보드 좌/우 네비게이션

### 5. StatsCounter
숫자가 카운트업되는 애니메이션 통계.
- Intersection Observer로 뷰포트 진입 시 트리거
- `useSpring`으로 부드러운 카운트업
- 한 번만 실행 (재진입 시 즉시 최종값)

### 6. HeroSection
풀스크린 패럴랙스 히어로 배너.
- 배경 이미지 패럴랙스 (`useScroll` + `useTransform`)
- 제목 텍스트 페이드인 + 슬라이드업
- 스크롤 인디케이터 (아래로 스크롤 유도)
- 모바일에서 패럴랙스 비활성화 (성능)

### 7. NavigationBar
반응형 네비게이션.
- 스크롤 다운 시 축소 + 배경 블러
- 스크롤 업 시 확장
- 모바일: 햄버거 → 풀스크린 오버레이 메뉴
- 현재 섹션 인디케이터
- 다크모드 토글 버튼

### 8. RecordCard
복원 사례 카드 컴포넌트.
- 호버 시 이미지 줌 + 오버레이 정보
- 카테고리 뱃지 (색상 구분)
- 지원유형 태그
- 클릭 시 상세 페이지 링크 (`Link` 컴포넌트)

### 9. SearchBar
실시간 검색.
- 디바운스된 입력 (300ms)
- 검색 결과 드롭다운 (최대 5개)
- 키보드 네비게이션 (화살표 + Enter)
- 검색어 하이라이팅
- `searchParams`와 동기화

## 글로벌 스타일 시스템

### CSS 변수 (다크모드)
```css
:root {
  --background: 0 0% 100%;
  --foreground: 0 0% 3.9%;
  --primary: 24 100% 50%;       /* 전통 금색 */
  --accent: 0 72% 51%;          /* 적색 포인트 */
  --card: 0 0% 98%;
  --border: 0 0% 89.8%;
  --radius: 0.75rem;
}

.dark {
  --background: 0 0% 6%;
  --foreground: 0 0% 98%;
  --card: 0 0% 9%;
  --border: 0 0% 18%;
}
```

### 폰트
- 한글: Pretendard (Variable)
- 영문/숫자: Inter (Variable)
- 로딩: `next/font/local` 또는 `next/font/google`

## 반응형 breakpoints
- `sm`: 640px (대형 모바일)
- `md`: 768px (태블릿)
- `lg`: 1024px (작은 데스크탑)
- `xl`: 1280px (데스크탑)
